Classpublic class Resultset
InheritanceResultset Object

A structXML resultset object. This is the structure exchanged between all semantic components. This data structure is composed of subjects (records) and each subject of a resultset is described using attribute/values. A resultset contains the description of records that are outputed by a system (most likely, a structWSF instance).
