Classpublic class Schema
InheritanceSchema flash.events.EventDispatcher

Conceptual structure used to drives all the semantic component. This structure is created from a irXML schema file. It describes all the attributes and types used to describe the subjects of a resultset.

getComparableAttributes()method 
public function getComparableAttributes(attribute:String, comparableAttributes:Array, inference:Boolean = true):void

Get all attributes that can be compared to a target attribute. Comparable means that their value should be analizable together (semantically comparable).

Parameters

attribute:String — URI of the attribute. Can in its full URI form like "http://foo.com/bar/..." or its internal abbreged form such as "foo_Bar...".
 
comparableAttributes:Array — All comparable attributes with the target SchemaAttribute
 
inference:Boolean (default = true) — Enable inference on the schema structure to have all possible comparable attributes

public function getSubAttributes(attributeID:String, recursively:Boolean = false):Array

Get all sub-types of a given type ID.

recursively:Boolean (default = false) — If this parameter is true, it will recursively browse the type structure to get all sub-types of this type. Otherwise, it will only get the immediate sub-types.

Returns
Array — An array of all the sub-types of the target type ID.
