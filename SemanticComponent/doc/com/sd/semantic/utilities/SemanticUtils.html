public static function getBoundAttributes(children:Array):Array

Get the list of all attributes bound to specific semantic controls in an application layout. Note: this function is normally used with the getChildrenRecur function to get all these bount attributes.

Parameters

children:Array — List of components for which we want to know if they are bound to any attribute.

Returns
Array — Returns a list of attribute URI bound to different components.
